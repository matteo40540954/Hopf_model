In this project for the class "Computational Neurology" we use a Hopf model, a model that simulates brain activity over time. 
According to this model every brain region is simplie in the form of an oscillator and a parameter (e.g., bifurcation parameter)
describes every brain oscillation that goes from a stable state (with no oscillations) and a limit-cycle state (sustained oscillations). 

Hopf model is a phenomenological model etc etc
